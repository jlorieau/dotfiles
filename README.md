Plugins
=======

Install
-------

```shell
#  git submodule add --force --depth=1 <repo_url> <path> 
$  git submodule add --force --depth=1 https://github.com/romkatv/powerlevel10k zsh/plugins/powerlevel10k 
```

Update
------

```shell
$ git sync
```

Delete
------

```shell
$ git submodule rm -f <path>
```
